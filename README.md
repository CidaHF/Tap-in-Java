Tap in Java
===========

The Tap class for Java.

Usage:

```
public static void main(String[] args)
{
    Tap tap = new Tap();

    tap.pressure.increase();
    tap.pressure.increase();
    tap.pressure.increase();
    tap.pressure.increase();
    tap.pressure.increase();
    tap.pressure.increase();
    tap.pressure.increase();
    tap.pressure.increase();
}
```

Outputs:
```
Tap: I'm a tap!
Tap: Catch me if you can!
Tap: Bang!
```
